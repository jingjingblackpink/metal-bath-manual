使用说明书

一、手动模式操作

  1. 在手动设置页面设定目标温度（4~100°C）和保温时间
  2. 选择速率模式（基础/高速/极限）
  3. 点击"运行"开始控温
  4. 到达设定温度后自动开始保温倒计时，倒计时结束蜂鸣提示并自动停止
  5. 运行中可随时点击"暂停"或"停止"

二、程控模式操作

  1. 在程序编辑页面编辑程序（最多5个程序，每个最多3步）
  2. 在程序控制页面选择程序并点击"运行"
  3. 每个步骤自动执行，全部完成蜂鸣提示自动停止

三、恢复室温

点击"恢复室温"按钮，设备自动控温至23°C后停止。适用于实验结束后安全取出样品。

运行中点击"停止"可中断，停止后风扇继续工作10秒。

功能说明

五、速率模式

模式升温速率降温速率适用场景
基础模式约3°C/min约2°C/min对速率要求不高
高速模式约5°C/min约3°C/min快速到达目标温度
极限模式不限制，以最快速度到达极限升降温需求

六、报警类型

温度传感器故障 温度异常 升降温异常 模块未正确安装 通信故障 存储器故障

设备运行中如出现异常，会自动停止并在报警页面显示信息。

⚠️ 温馨提示

1. 关闭仪器前，请先停止设备运行，等待仪器温度恢复至室温后，再关闭电源开关。

2. 切勿在关闭电源后立即重新开机，不当操作将影响仪器的使用寿命。
